Platform

Everything You Need

A complete end-to-end inspection platform built for civil engineers and field teams.

🤖

AI Defect Detection

YOLO26 detects cracks, potholes, and manholes with bounding boxes and confidence scores in seconds.

📊

PCI Scoring

Density-based logarithmic Pavement Condition Index (0–100) computed automatically per inspection.

🎚️

Confidence Threshold

Adjustable detection sensitivity per inspection from 15% to 75% — balance precision vs recall.

🖼️

Before/After Slider

Drag to compare original and AI-annotated imagery side by side. Highlights clip to the annotated side only.

🔍

Detection Highlighting

Click any detection to highlight its bounding box with a glowing overlay directly on the image.

🗂️

Image Gallery

Grid view of all inspection images with defect count badges, lightbox, and compare shortcuts.

✅

Engineer Review

Confirm, flag, or dismiss each AI detection before finalizing. Every action is attributed to the reviewer.

📝

Engineer Notes

Add inline notes per detection. Saved instantly, displayed below the card, included in PDF reports.

🔄

Bulk Re-analyze

Re-run YOLO on an existing inspection with a new threshold without creating a new inspection.

📄

PDF Reports

Branded reports with annotated images, PCI score, defect tables, engineer notes, and attribution.

📋

CSV Export

Structured data export with all detection metadata, bbox coordinates, and actioned-by tracking.

🗺️

Map View

Auto-geocoded project pins on OpenStreetMap, color-coded by PCI score for instant spatial overview.

📈

Analytics Dashboard

Defect distribution pie, PCI trend area chart, defects per inspection bar chart, status breakdown.

👥

Multi-user Auth

JWT-based login with three roles. Full activity attribution — every action tracked to the responsible user.

🏢

Team Collaboration

Shared projects across the team. Multiple engineers can work on the same inspection simultaneously.

Detection

Defect Classes

Three categories detected automatically, each with a severity weight that feeds into the PCI score.

🕳️

Pothole

Surface depressions posing immediate safety risk. Highest severity in the scoring model.

HIGH · weight × 1.0
⚡

Crack

Surface cracking indicating structural stress, fatigue, or water infiltration damage.

MEDIUM · weight × 0.4
🔘

Manhole

Manhole cover detection for infrastructure mapping and surface condition logging.

LOW · weight × 0.1
Scoring

Pavement Condition Index

PCI is a numerical rating from 0 (failed) to 100 (excellent), calculated using a density-based logarithmic deduction model.

Scoring Formula

PCI = 100 − Σ ( 30 × log(1 + densitytype) × weighttype )

pothole × 1.0 crack × 0.4 manhole × 0.1

Access Control

Role-Based Access

Three roles designed for real civil engineering team workflows.

Technicien

Field Technician

  • Create new inspections
  • Upload road imagery
  • View AI results
  • Access all projects
Engineer

Field Engineer

  • All Technicien permissions
  • Confirm / flag / dismiss detections
  • Add engineer notes per detection
  • Export PDF & CSV reports
  • Mark inspections complete
Manager

Project Manager

  • All Engineer permissions
  • Manage team members
  • Change user roles
  • Enable / disable accounts
  • Full platform oversight
